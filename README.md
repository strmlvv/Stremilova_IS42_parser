# Парсер судебных решений

Проект для парсинга судебных решений с использованием Python, регулярных выражений и SQLite.

## Функции
- Парсинг Word-файлов (.docx)
- Извлечение:
  - номера дела
  - даты решения
  - судьи
  - места рассмотрения
- Сохранение данных в SQLite

## Используемые технологии
- Python
- SQLAlchemy
- python-docx
- Regular Expressions
- SQLite
